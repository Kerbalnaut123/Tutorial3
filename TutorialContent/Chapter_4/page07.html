Tutorial - My First Game

Getting to grips with the room editor is essential if you want to get the most out of GameMaker Studio 2, and there are many, many features to it. In this chapter we've covered two of the main ones, the layer system - and in particular tilemap layers - and the camera views system. Let's quickly cover the main points you should have picked up while going through this:

  • How to set up a tile set resource
  • How to create and order layers in the room editor
  • How to paint tiles into a tilemap layer
  • How to enable and set up view cameras
  • How to make a view camera follow an object instance

Our arena shooter game is starting to look pretty good now, and we have all the essentials in there. However it's missing one or two things to take it from a prototype to a finished game, one of which is a scoring system which is what we'll cover in the next chapter of this tutorial...
