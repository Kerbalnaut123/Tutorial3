Tutorial - My First Game

After completing this Chapter we now have a nicer introduction to the game. While setting this up you should have learned the following:

  • How to add and re-order rooms in the resource tree
  • How to change the built-in properties (like image alpha) for drawing a sprite
  • How to use the keyboard events
  • How to change rooms using the Room actions
With that we are almost finished with this tutorial, but we have one final task before us. Currently the game has a limited number of enemies to destroy, which limits the gameplay greatly, so in the next chapter we'll look at "spawning" more enemies as you play...
